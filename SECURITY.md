# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by emailing **security@ottersight.com**. Include:

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

You will receive a response within 48 hours. We aim to release a fix within 7 days of confirmation.

## Disclosure Policy

We follow [responsible disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure). Once a fix is released, we will publish a security advisory on GitHub.
